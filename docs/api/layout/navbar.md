# Navbar

导航栏

## 示例

```vue
<Navbar theme="primary">primary</Navbar>
```

## Props

| 名称       | 类型     | 默认值           | 可选值 | 说明         |
| ---------- | -------- | ---------------- | ------ | ------------ |
| backHandle | Function | `history.go(-1)` |        | 返回事件回调 |

## slot

| 名称  | 说明     |
| ----- | -------- |
| left  | 左侧插槽 |
| right | 右侧插槽 |
